## 1，自动安装

1. 克隆项目并进入目录

```
git clone https://github.com/bozoyan/AsrTools.git
cd AsrTools
```

1. 安装依赖

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
